# Night Highway Survival

Night Highway Survival is a 2D car dodging game developed using C++ and SFML. The player controls a car on a highway and must avoid collisions with incoming enemy vehicles while surviving as long as possible.

The game features real-time player movement, collision detection, dynamic score tracking, increasing difficulty levels, and texture-based graphics. It is designed using Object-Oriented Programming (OOP) principles with separate classes for player and enemy vehicles, making the code modular and maintainable.

## Features

* Player-controlled vehicle movement
* Random enemy car spawning
* Collision detection and Game Over system
* Dynamic score and survival time tracking
* Gradually increasing game speed and difficulty
* Start screen, restart functionality, and quit option
* Texture-based graphics using SFML
* Object-Oriented Design (Inheritance and Classes)

## Technologies Used

* C++
* SFML (Simple and Fast Multimedia Library)
* Object-Oriented Programming (OOP)

## Controls

* Enter → Start Game
* Left / Right Arrow Keys → Move Vehicle
* R → Restart Game
* ESC → Exit Game

This project was developed as part of an academic game programming project to demonstrate practical implementation of C++ programming, graphics rendering, event handling, and game development concepts.
